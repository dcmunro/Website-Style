Text Style

These are just a few examples of elements I use and how I style them

H1: Main heading

  • Font-size: 40px
  • Font-weight: 700
  • Text-Decoration / Font Style: none

H4: A Variation of a Sub Heading

  • Font-size: 30px
  • Font-weight: 500
  • Text-Decoration / Font Style: Underlined / None
Span: Used for inline additions and/or modifications
  • Font-size: 16-20px
  • Font-weight: 300-500
  • Text-Decoration / Font Style: Italicized / Underlined / None
